  • Honeycrisp

    GitHub Link

    Honeycrisp is my current side-project! It originally started as an alternative online interface for Apple Music, but once I switched back to Spotify I found I had different needs. It is created entirely with Angular; there is, currently, no need for a backend, although that may change!
